## 📱 コミュニケーション体制

### **主要コミュニケーションツール**

| ツール                 | 主な利用者         | 用途                         |
| ---------------------- | ------------------ | ---------------------------- |
| **LINE**               | 全社員・職人さん   | 日常連絡・進捗報告・緊急連絡 |
| **AGO-グループ Gmail** | 南さん・篠宮さん   | 発注メール管理・公式連絡     |
| **アンドパッド**       | 現場チーム         | 現場管理・進捗管理           |
| **スプレッドシート**   | 南さん・事務チーム | ステータス管理・記録         |

### **情報フロー**

```
発注依頼 → 篠宮さん（LINE受信）
↓
メール作成 → サンセイ・はざいや（CC: AGO-Gmail）
↓
確認・記録 → 南さん（AGO-Gmail + スプレッドシート）
↓
進捗管理 → 職人さん（LINE報告）→ 南さん（ステータス更新）
```
